1.йҳ…иҜ»дёӢйқўз»ҷеҮәзҡ„ DM зҡ„иҫ“е…ҘзӨәдҫӢдёӯпјҲзӨәдҫӢ DM е®№йҮҸдёә 4KBпјҢеҚі 32bit Г— 1024еӯ—пјүпјҢж №жҚ®дҪ зҡ„зҗҶи§Јеӣһзӯ”пјҢиҝҷдёӘ addr дҝЎеҸ·еҸҲжҳҜд»Һе“ӘйҮҢжқҘзҡ„пјҹең°еқҖдҝЎеҸ· addr дҪҚж•°дёәд»Җд№ҲжҳҜ [11:2] иҖҢдёҚжҳҜ [9:0] пјҹ

вҖӢ иҝҷдёӘaddrдҝЎеҸ·жҳҜALUзҡ„з»“жһңпјҢдҪңдёәDMзҡ„ең°еқҖпјҢз”ұдәҺDMжҳҜд»Ҙеӯ—зј–еқҖпјҢжҜҸеӣӣдёӘеӯ—иҠӮзј–дёҖдёӘең°еқҖпјҢиҖҢALUresultи®Ўз®—еҮәжқҘзҡ„ең°еқҖжҳҜд»Ҙеӯ—иҠӮзј–еқҖпјҢжүҖд»ҘжҚўз®—жҲҗеӯ—ең°еқҖйңҖиҰҒжҜҸеӣӣдҪҚеҸ–дёҖдҪҚең°еқҖпјҢеҚід»ҺALUresultзҡ„з¬¬дёүдҪҚејҖе§ӢеҸ–ең°еқҖпјҢеӣ дёәDMзҡ„е®№йҮҸжҳҜ4KBпјҢжүҖд»ҘжҢүз…§еӯ—иҠӮзј–еқҖжҳҜ12дҪҚең°еқҖпјҢеҸ–ALUresult[11:0] , д»Ҙеӯ—еҒҡеҚ•дҪҚе°ұжҳҜ10дҪҚең°еқҖпјҢдҫҝжҳҜеҸ–10дҪҚпјҢеҚі[11:2]пјҢеҗҢж—¶д»Ҙ[11:2]дҪңдёәең°еқҖеҸҜд»ҘжҸҗйҶ’дҪҝз”ЁиҖ…пјҢз»ҷDMзҡ„addrеј•и„ҡжҺҘе…Ҙзҡ„жҳҜALUresultзҡ„[11:2]дҪҚгҖӮ

2.жҖқиҖғдёҠиҝ°дёӨз§ҚжҺ§еҲ¶еҷЁи®ҫи®Ўзҡ„иҜ‘з Ғж–№ејҸпјҢз»ҷеҮәд»Јз ҒзӨәдҫӢпјҢе№¶е°қиҜ•еҜ№жҜ”еҗ„ж–№ејҸзҡ„дјҳеҠЈгҖӮ

жҢҮд»ӨеҜ№еә”зҡ„жҺ§еҲ¶дҝЎеҸ·еҰӮдҪ•еҸ–еҖј

иҝҷдёҖз§ҚиҜ‘з Ғж–№ејҸпјҢеҜ№дәҺдҝЎеҸ·зҡ„жҺ§еҲ¶дёҚе®№жҳ“йҒ—жјҸпјҢеҜ№дәҺжҜҸдёҖдёӘдҝЎеҸ·йғҪйңҖиҰҒз»ҷдёҖдёӘеҖјпјҢжё…жҷ°жҳ“иҜ»жҜ”иҫғзӣҙи§ӮпјҢдҪҶжҳҜж·»еҠ жҢҮд»ӨжҜ”иҫғеӨҚжқӮпјҢйңҖиҰҒз»ҷеҮәе®Ңж•ҙзҡ„жҺ§еҲ¶дҝЎеҸ·пјҢеҜ№дәҺдёҚйңҖиҰҒзҡ„дҝЎеҸ·д№ҹйңҖиҰҒз»ҷе®ҡй»ҳи®ӨеҖјпјҢиҝҷж ·д№ҹдјҡеҜјиҮҙжҺ§еҲ¶йғЁеҲҶзҡ„д»Јз ҒжҜ”иҫғй•ҝгҖӮ

жҺ§еҲ¶дҝЎеҸ·жҜҸз§ҚеҸ–еҖјжүҖеҜ№еә”зҡ„жҢҮд»Ө

иҝҷдёҖз§ҚиҜ‘з Ғж–№ејҸпјҢжҳҜеҜ№жҺ§еҲ¶дҝЎеҸ·з”ЁдәҶжҲ–жҢҮд»Өзҡ„ж–№ејҸпјҢеҰӮжһңж»Ўи¶іиҝҷжқЎжҢҮд»ӨпјҢе°ұдјҡдҪҝзҡ„жҺ§еҲ¶дҝЎеҸ·жңүж•ҲпјҢиҝҷз§Қж–№ејҸзҡ„дјҳзӮ№еңЁдәҺеҸҜд»ҘеҫҲе®№жҳ“зҡ„ж·»еҠ жҢҮд»ӨпјҢеҜ№дәҺжҢҮд»ӨеҸӘйңҖиҰҒеңЁзӣёеә”жҺ§еҲ¶дҝЎеҸ·д№ӢеҗҺжҲ–дёҠдёҖдёӘеҚіеҸҜпјҢдҪҶжҳҜзјәзӮ№жҳҜдёҚеӨҹзӣҙи§ӮпјҢеҸҜиғҪдјҡйҖ жҲҗжјҸеҠ дҝЎеҸ·зҡ„й”ҷиҜҜгҖӮ

3.еңЁзӣёеә”зҡ„йғЁд»¶дёӯпјҢеӨҚдҪҚдҝЎеҸ·зҡ„и®ҫи®ЎйғҪжҳҜеҗҢжӯҘеӨҚдҪҚпјҢиҝҷдёҺ P3 дёӯзҡ„и®ҫи®ЎиҰҒжұӮдёҚеҗҢгҖӮиҜ·еҜ№жҜ”еҗҢжӯҘеӨҚдҪҚдёҺејӮжӯҘеӨҚдҪҚиҝҷдёӨз§Қж–№ејҸзҡ„ reset дҝЎеҸ·дёҺ clk дҝЎеҸ·дјҳе…Ҳзә§зҡ„е…ізі»гҖӮ

вҖӢ еңЁеҗҢжӯҘеӨҚдҪҚдёӯпјҢclkзҡ„дјҳе…Ҳзә§жҳҜй«ҳдәҺresetзҡ„пјҢеҸӘжңүеңЁж—¶й’ҹдёҠеҚҮжІҝеҲ°жқҘж—¶resetдҝЎеҸ·жңүж•ҲжүҚиҝӣиЎҢеӨҚдҪҚпјҢеҚ•жңүresetдҝЎеҸ·иҖҢж—¶й’ҹдёҠеҚҮжІҝдҝЎеҸ·жІЎжңүеҲ°жқҘдёҚиғҪиҝӣиЎҢеӨҚдҪҚгҖӮд»Ҙverilogд»Јз ҒжқҘиҜҙпјҢresetдҝЎеҸ·жҳҜеңЁclkзҡ„и§ҰеҸ‘дёӯзҡ„пјҢеҸӘжңүеӨ–еұӮжңүж•ҲеҶ…еұӮжүҚиғҪжңүж•ҲгҖӮ

еңЁејӮжӯҘеӨҚдҪҚдёӯпјҢclkе’Ңresetзҡ„дјҳе…Ҳзә§жҳҜзӣёеҗҢзҡ„пјҢдёҚи®әжҳҜclkе’ҢresetйғҪиғҪи§ҰеҸ‘жҺ§еҲ¶пјҢеҚіеңЁд»»дҪ•ж—¶еҖҷеҸӘиҰҒresetдҝЎеҸ·жңүж•ҲпјҢйғҪиғҪи§ҰеҸ‘еӨҚдҪҚпјҢд»Ҙverilogд»Јз ҒжқҘиҜҙпјҢж•Ҹж„ҹжқЎд»¶жҳҜposedge clkжҲ–иҖ…posedge resetпјҢеҸӘиҰҒдәҢиҖ…ж»Ўи¶іе…¶дёӯдёҖдёӘжғ…еҶөпјҢе°ұеҸҜд»ҘиҝӣиЎҢжҺҘдёӢжқҘзҡ„ж“ҚдҪңгҖӮ

4.C иҜӯиЁҖжҳҜдёҖз§Қејұзұ»еһӢзЁӢеәҸи®ҫи®ЎиҜӯиЁҖгҖӮC иҜӯиЁҖдёӯдёҚеҜ№и®Ўз®—з»“жһңжәўеҮәиҝӣиЎҢеӨ„зҗҶпјҢиҝҷж„Ҹе‘ізқҖ C иҜӯиЁҖиҰҒжұӮзЁӢеәҸе‘ҳеҝ…йЎ»еҫҲжё…жҘҡи®Ўз®—з»“жһңжҳҜеҗҰдјҡеҜјиҮҙжәўеҮәгҖӮеӣ жӯӨпјҢеҰӮжһңд»…д»…ж”ҜжҢҒ C иҜӯиЁҖпјҢMIPS жҢҮд»Өзҡ„жүҖжңүи®Ўз®—жҢҮд»ӨеқҮеҸҜд»ҘеҝҪз•ҘжәўеҮәгҖӮ иҜ·иҜҙжҳҺдёәд»Җд№ҲеңЁеҝҪз•ҘжәўеҮәзҡ„еүҚжҸҗдёӢпјҢaddi дёҺ addiu жҳҜзӯүд»·зҡ„пјҢadd дёҺ addu жҳҜзӯүд»·зҡ„гҖӮжҸҗзӨәпјҡйҳ…иҜ»гҖҠMIPS32В® Architecture For Programmers Volume II: The MIPS32В® Instruction SetгҖӢдёӯзӣёе…іжҢҮд»Өзҡ„ Operation йғЁеҲҶгҖӮ

addiдёҺaddiuзҡ„еҢәеҲ«еңЁдәҺпјҢеҪ“еҮәзҺ°жәўеҮәж—¶пјҢaddiuе°ҶеҝҪз•ҘжәўеҮәпјҢе°ҶжәўеҮәзҡ„жңҖй«ҳдҪҚиҲҚејғпјӣaddiдјҡжҠҘе‘ҠSignalException(IntegerOverflow)гҖӮ

еҰӮжһңеҝҪз•ҘжәўеҮәпјҢaddiдёҚдјҡжҠҘй”ҷпјҢдәҢиҖ…зӯүд»·гҖӮaddе’ҢadduеҗҢзҗҶгҖӮ

P4иҜҫдёҠеӣһеҝҶ

з»„еҗҲйҖ»иҫ‘еҲ«з”Ёйқһйҳ»еЎһиөӢеҖјпјҢдјҡжӯ»зҡ„гҖӮгҖӮ

еңЁз»„еҗҲйҖ»иҫ‘дёӯдҪҝз”ЁеҜ„еӯҳеҷЁи®°еҫ—жҜҸж¬ЎдҪҝз”ЁеүҚиҰҒжё…йӣ¶ дёҚз„¶дјҡеҜјиҮҙй”ҷиҜҜгҖӮ

йҖ»иҫ‘еҸіз§»

жІЎжқҘеҫ—еҸҠзңӢзҡ„з¬¬дёҖйҒ“йўҳ

ras

ifпјҲimm == 0пјүGPR[rt] <= GPR[rs]
else if (imm < 32)
temp = пҪһGPR[rs][31:32-imm] || GPR[rs][31-imm:0]
num = how many 1 in imm
GPR[rt] = temp >>> num
else
temp = ~GPR[rs]
GPR[rt] = temp >>> num

иҰҒз»ҷImmеҲӨж–ӯеӨ§е°ҸпјҢиІҢдјјдёҚиғҪеҲӨж–ӯ i >= дёҖдёӘиҙҹж•°зҡ„жғ…еҶө
